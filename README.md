# Guides

This repository contains simple guides for content contributors.

## Available Guides

### [How to Send Blog Posts](simple-guide_how-to-send-blog-posts.md)

A step-by-step guide for writing and submitting blog posts in Markdown format. Covers:

- What software to use (Mac and Windows)
- How to structure your posts and images folders
- Markdown formatting basics (headings, bold, links, images)
- How to zip and send your files

### [Optional Layout Tags for Images](optional-layout-tags-for-images.md)

A guide for controlling image alignment within blog posts. Covers:

- How to place an image on the left or right of text
- Tag syntax and placement rules
- When tags are optional vs. required
