# Optional Layout Tags (For Images)

Sometimes you may want the **image aligned beside the text**.

You can add a **layout tag above the image**.

If you do **not add a tag**, the image will appear normally.

---

# Image on the Left

Write this line **above the image**:

```
[image-left]
```

Example:

```
## Growing Tulips

[image-left]

![Pink tulips](pink_tulips.png "Pink tulips")

Tulips are one of the most popular spring flowers. They grow easily and bring bright colour to gardens.
```

Result:

* Image appears **on the left**
* Text appears **beside the image**

---

# Image on the Right

```
[image-right]
```

Example:

```
## Growing Tulips

[image-right]

![Pink tulips](pink_tulips.png "Pink tulips")

Tulips grow best in well-drained soil and sunny areas.
```

---

# Important Rules

1. The layout tag must be **above the image**
2. Write the tag **on its own line**
3. Leave **one empty line before the image**
4. Tags are **optional**

If you do not add a tag, the image will appear in the normal layout.

---

# Example Section

```
## Growing Tulips

[image-left]

![Pink tulips](pink_tulips.png "Pink tulips")

Tulips are easy to grow and add colour to spring gardens.
```
