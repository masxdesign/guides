# Simple Guide: How to Send Blog Posts

Please send your blog posts using **Markdown (.md)** along with the images.

You will send **two zipped files**:

* `posts.zip`
* `images.zip`

The steps below explain exactly how to prepare everything.

---

# 1. What Software To Use

You can write the blog using a simple text editor.

## Mac

Use one of these:

* **TextEdit** (already installed)
* **Visual Studio Code** (free)

### If using TextEdit

1. Open **TextEdit**
2. Click **Format → Make Plain Text**
3. Write your blog
4. Click **File → Save**
5. Save the file with **.md** at the end

Example:

```
post_1.md
```

---

## Windows

Use one of these:

* **Notepad** (already installed)
* **Notepad++**
* **Visual Studio Code**

### If using Notepad

1. Open **Notepad**
2. Write the blog
3. Click **File → Save As**
4. Save the file as:

```
post_1.md
```

---

# 2. Create the Posts Folder

Create a folder called:

```
posts
```

Put **all blog files inside this folder**.

Example:

```
posts/
    post_1.md
    post_2.md
    post_3.md
```

The filename **does not have to match the blog title**.

---

# 3. Create the Images Folder

Create another folder called:

```
images
```

Inside it, create **one folder for each blog post**.

Each folder must have the **same name as the Markdown file (without `.md`)**.

Example:

```
images/
    post_1/
    post_2/
```

---

# 4. Add Images

Put the images for that blog post inside its folder.

Example:

```
images/
    post_1/
        pink_tulips.png
        yellow_daffodils.png
```

Use **underscores instead of spaces** in image filenames.

Good:

```
pink_tulips.png
spring_garden_flowers.jpg
```

Not good:

```
pink tulips.png
```

---

# 5. Writing the Blog

## Blog Title

Start with the **main title** using one `#`.

Example:

```
# Beautiful Spring Flowers
```

---

## Sections (like Heading 2 in Word)

Main sections in your blog use **two `#` symbols**.

This works like **Heading 2 in Microsoft Word**.

Example:

```
## Popular Spring Flowers
```

---

## Smaller Sections (like Heading 3 in Word)

Subsections inside a section use **three `#` symbols**.

This works like **Heading 3 in Microsoft Word**.

Example:

```
### Growing Tips
```

---

## Example Structure

```
# Beautiful Spring Flowers

## Popular Spring Flowers

### Tulips

### Daffodils

## Growing Tips

### Soil Preparation

### Watering Advice
```

This structure works like:

* Title
* Heading 2
* Heading 3

in **Microsoft Word**.

---

# 6. Bold Text

To make text bold, use **two stars**.

Example:

```
Spring is the **best season for colourful flowers**.
```

---

# 7. Links

To add a link:

```
[Link Text](https://example.com)
```

Example:

```
Learn more here:
[Royal Horticultural Society](https://www.rhs.org.uk)
```

---

# 8. Images in the Blog

Use this format:

```
![alt text](image_name.png "image title")
```

Example:

```
![Pink tulips in spring](pink_tulips.png "Pink tulips")
```

Important:

* The image must be inside the **correct image folder**
* The **filename must match exactly**

---

# 9. Example Blog File

Example file:

```
post_1.md
```

Example content:

```
# Beautiful Spring Flowers

Spring is the **perfect season for colourful gardens**.

## Popular Spring Flowers

Tulips and daffodils are some of the most loved spring flowers.

![Pink tulips in spring](pink_tulips.png "Pink tulips")

### Why Gardeners Love Them

They are easy to grow and look beautiful.

Learn more here:
[Royal Horticultural Society](https://www.rhs.org.uk)

![Yellow daffodils blooming](yellow_daffodils.png "Yellow daffodils")
```

---

# 10. Final Folder Structure

Your files should look like this before zipping:

```
posts/
    post_1.md
    post_2.md

images/
    post_1/
        pink_tulips.png
        yellow_daffodils.png

    post_2/
        red_roses.png
```

---

# 11. Zip the Folders

Zip **each folder separately**.

### Zip the Posts Folder

Right-click the `posts` folder and compress it.

You will get:

```
posts.zip
```

---

### Zip the Images Folder

Right-click the `images` folder and compress it.

You will get:

```
images.zip
```

---

# 12. What To Send

Please send these two files:

```
posts.zip
images.zip
```
